# 💸 PayCab — Personal Transaction Tracker

A simple, personal web-based system for **tracking freelance payments, expenses, and client transactions in one place**.

Built with **HTML, CSS, and JavaScript** — lightweight, fast, and easy to manage.

---

## 🚀 Live Demo

* 🌐 Try it here: **[https://paycab.netlify.app/](https://paycab.netlify.app/)**

---

## 📌 Features

### 👤 About Section

* Simple introduction of the user
* Personal profile display for freelance identity

### 🛠️ Services Section

* Lists freelance services offered
* Clean and structured layout for client visibility

### 💳 Payment Tracking

* Manual tracking of income and expenses
* Helps organize freelance payments clearly
* Supports external or manual payment handling

### ⭐ Rating / Feedback System

* Displays service ratings
* Collects or shows client feedback

### 🔍 User Interface

* Simple and clean layout
* Easy navigation and minimal design
* Built for personal productivity

---

## 🧠 Purpose

PayCab is designed to:

* Track freelance income and expenses
* Organize client payments and due status
* Keep everything structured in a single dashboard
* Avoid manual spreadsheet tracking

---

## 🚀 Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)
* Netlify (deployment)

---

## 🧱 Project Files

### `index.html`

* Main structure of the PayCab system
* Contains all sections (About, Services, Payments, Ratings)

### `styles.css`

* Layout styling
* Simple UI design for readability and clarity

### `script.js`

* Handles basic interactions
* Manages UI behavior and dynamic elements

## 📝 Usage

### For Users

1. Open the live site
2. Navigate through sections
3. Add or manage payment-related information manually
4. Use it as a personal tracking dashboard

```css id="paycab1"

  --primary-color: #2ecc71;

```

#### ⚙️ Modify Content

* Edit services in `index.html`
* Adjust layout in `styles.css`
* Add functionality in `script.js`
